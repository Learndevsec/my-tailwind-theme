# My Tailwind Theme

A modern WordPress theme built with Tailwind CSS.

## Installation

1. Install dependencies:
```bash
   npm install
```

2. Build Tailwind CSS:
```bash
   npm run build
```

3. Upload the theme folder to `/wp-content/themes/` or zip and upload via WordPress admin.

## Development

Run Tailwind in watch mode:
```bash
npm run dev
```

## Features

- Tailwind CSS 3.4
- Responsive mobile menu
- Widget-ready sidebar
- Custom logo support
- Featured images
- Translation ready
